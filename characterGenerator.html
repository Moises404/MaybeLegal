Photoshop Tools

This is a really basic interactive tutorial on the basic photoshop tools that will be used in for the character generator. Click on each highlighted tool to reveal their function.

Marquee

drag and hold different parts of the a layer to move.

Selector

tool used to geometrically select a portion of an image.

Lasso

three different type of Lassos are magnetic, polygonal, and regular. Used Shift-L to toggle between them.

Magic Wand

selects portions of the layers that are similar. The tolerance of similarity can be changed in the top of the applications.

Crop

used to change the size of the canvas, We will used this tool to crop each portion of the characters parts.

Eye-drop

used to select a specific color of an image. Hot key command i.

Heal-brush

tries to even out a portion of the image.

Brush

basic drawing tool, used after lasso or magic wand to paint a specific portion of an image. Hot key to increase size is left and right brackets.

Clone Stamp

selects a point on an image using option-click and uses it on the next portion of an image you click on. 'Cloning those pixels.'

Eraser

basic eraser, used after lasso or magic wand to erase a specific portion of an image. Hot key to increase size is left and right brackets.

Saving Png24 Files

png24s are transparent images that that serve well on in web platforms. In photoshop to save an image as a png24 on mac shift + option + command + s, or go to menu and click save as web. Once there select png 24 and save the file for that specific character body part.
